# 📝 iTask - Todo App

iTask is a simple and clean Todo application that helps you manage your daily tasks efficiently in one place.

🔗 **Live Demo:** https://itaskatodoapp.netlify.app/

---

## 🚀 Features

- ➕ Add new tasks easily  
- ✅ Mark tasks as completed  
- 👀 Toggle to show/hide finished tasks  
- ✏️ Edit existing tasks  
- 🗑️ Delete tasks  
- 💾 Persistent storage (tasks remain after refresh)

---

## 🛠️ Tech Stack

- ⚛️ React (with Vite)
- 🎨 Tailwind CSS
- 🧩 UUID (for unique task IDs)

---

## 📦 Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/itask.git

# Navigate to project folder
cd itask

# Install dependencies
npm install

# Run the development server
npm run dev
